# instructions:

- once the application is booted up create a new work space click the plus icon in the bottom left
- once the work space is open click the plus icon again to import pdf's
- there wont be anything there for now. shift select all of the PDF's within the readme folder and drag them to the emulator
- move to downloads within the android file explorer
- select all pdfs that you want to compare
- click select and you will be taken back to the application.
